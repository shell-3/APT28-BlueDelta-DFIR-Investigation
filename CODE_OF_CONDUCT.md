# Code of Conduct

## Our Pledge

This repository is maintained as a professional DFIR research resource. All participants — contributors, commenters, and users — are expected to uphold the highest standards of the security research community.

## Standards

**Expected behavior:**

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experience levels
- Gracefully accept constructive criticism
- Focus on what is best for the defensive security community
- Show empathy towards other community members
- Cite sources and give credit for others' work

**Unacceptable behavior:**

- Discussing or sharing techniques for unauthorized system access
- Posting content that facilitates harm to real systems or individuals
- Harassment, trolling, or personal attacks
- Publishing others' private information without consent
- Using this repository's content for illegal activities

## Security Research Ethics

This community adheres to responsible disclosure principles:

- **Authorization first:** Only test against systems you own or have explicit written permission to test
- **Responsible disclosure:** Report vulnerabilities through proper channels, not public posts
- **No harm:** Research that could cause harm to real systems, organizations, or individuals is prohibited

## Enforcement

Violations of this Code of Conduct may result in:

1. Warning (first offense)
2. Temporary ban from contributing
3. Permanent ban for serious or repeated violations

## Reporting

Report violations privately through GitHub's reporting feature or by contacting the repository maintainer.
